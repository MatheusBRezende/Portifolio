# Portfólio Pessoal 🌐

Este é o meu portfólio pessoal, criado para exibir meus projetos, habilidades e experiências na área de desenvolvimento web. Desenvolvido com tecnologias modernas, ele reflete minha paixão por tecnologia e aprendizado contínuo.

## 🛠️ Tecnologias Utilizadas
- **HTML**: Estruturação das páginas
- **CSS**: Estilização e design responsivo
- **JavaScript**: Interatividade e animações
- **GitHub Pages**: Hospedagem do site

## ✨ Funcionalidades
- Exibição de informações pessoais e contatos
- Seção de projetos com links para os repositórios no GitHub
- Design responsivo para dispositivos móveis, tablets e desktops
- Navegação simples e intuitiva
